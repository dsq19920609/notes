#### 格式

~~~js
prettier [options] [file/dir/glob ...]
~~~

~~~js
// 格式化所有文件
prettier --write .
~~~

~~~js
prettier --single-quote --trailing-comma all --write docs package.json "{app,__{tests,mocks}__}/**/*.js"
~~~

<br/>

#### 常用的命令

~~~js
--wirte 

--check

----ignore-unknown

prettier "**/*" --write --ignore-unknown
~~~
